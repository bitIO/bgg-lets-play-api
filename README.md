# BGG Let's Play API

Project wrapping BGG XML API providing endpoints to obtain lists of games to be
played or to buy based on other users plays and collections.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
